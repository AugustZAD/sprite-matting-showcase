透明立绘抠图 · 多背景展示

已抠好的 RGBA 立绘叠在可切换的底色上，方便肉眼检查边缘。三方案并排： ToonOut（推荐） / MODNet（现状） / 规则·白描边。底色切到 黑看白边/halo，切到品红/青绿看残色与缺口，切到白看黑边自然度， 棋盘看真实透明；色板右侧三张是真实游戏背景（直接取自 Demo 场景图）， 这才是叠进成品时的真实效果。点任意图可全屏放大对比细节。

怎么看

  • 白边 / halo：底色切黑。规则法的白贴纸边在这里最明显；ToonOut/MODNet 应几乎无白边。
  • 残色 / 没抠净的缝隙：底色切品红或青绿（与多数发色对比强）。
  • 黑边自然度 / 暗发糊边：底色切白。
  • 真实透明：底色切棋盘——白/灰格就是真正透明的地方。
  • 真实游戏背景：色板最右三张（老公寓 / 天台夜 / 酒吧夜，来自 Demo 场景图）—— 立绘叠上去边缘是否自然、有没有白边/残色一眼可见，这是最贴近成品的检验。
  • 软边% 越低越锐、破洞% 越低越好；规则法零成本但带白边、撞色/半透明会崩。
